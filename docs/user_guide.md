# 在线刷题系统用户使用说明

## 1. 系统简介

在线刷题系统是一个基于IP自动识别用户身份的刷题平台，无需注册登录即可使用。系统支持多种题型练习和在线考试，帮助用户提升编程技能和知识水平。

## 2. 核心功能

### 2.1 仪表盘
- **学习概览**：显示累计刷题数、客观题正确率、连续学习天数等核心数据
- **级别进度**：展示不同级别（Entry/Work/Pro）的完成百分比
- **最近活动**：查看最近5条答题记录和状态

### 2.2 题库
- **按级别筛选**：支持Entry（入门级）、Work（进阶级）、Pro（专家级）三个级别
- **按科目筛选**：支持算法、语言、规范、设计四个科目
- **按题型筛选**：支持单选题、多选题、判断题、主观题四种题型
- **题目状态**：显示题目是否已做、正确或错误
- **详情查看**：点击题目查看完整详情和解析

### 2.3 考试
- **自定义考试**：选择级别、科目和题量（10/30/50/100题）
- **自动计时**：每道题2分钟，超时自动交卷
- **实时保存**：答案自动保存，防止数据丢失
- **题号导航**：快速跳转到指定题目，标记不确定题目
- **考试报告**：交卷后生成详细的考试报告，包括总分、各科目正确率和错题列表

### 2.4 错题本
- **自动记录**：自动记录客观题错题
- **筛选功能**：按科目和掌握状态筛选
- **消灭模式**：做对后减少错误次数，达到3次做对标记为已掌握
- **难度标识**：错误次数≥3的题目标记为"难"
- **导出功能**：支持导出为Markdown格式

### 2.5 排行榜
- **综合排行**：所有用户的综合得分排名
- **级别排行**：不同级别的用户排名
- **科目排行**：不同科目的用户排名
- **排名标识**：前三名用特殊标识突出显示

## 3. 使用流程

### 3.1 首次使用
1. 访问系统网址，系统会自动根据IP识别您的身份
2. 在顶部的IP警告条中可以看到您的身份识别码（IP地址）
3. 建议将该IP地址保存，以便在IP变动时联系管理员进行数据迁移

### 3.2 练习题目
1. 进入"题库"页面
2. 使用筛选条件找到感兴趣的题目
3. 点击题目查看详情
4. 尝试回答后查看答案和解析

### 3.3 参加考试
1. 进入"考试"页面
2. 选择考试级别、科目和题量
3. 点击"开始考试"按钮
4. 在考试界面中回答题目，可使用题号导航快速跳转
5. 完成后点击"交卷"按钮查看考试结果

### 3.4 复习错题
1. 进入"错题本"页面
2. 查看自动记录的错题
3. 点击"消灭"按钮标记已掌握的题目
4. 可导出错题本为Markdown格式进行离线复习

## 4. 级别与科目说明

### 4.1 级别划分
- **Entry级**：入门级，仅包含算法科目，适合初学者
- **Work级**：进阶级，包含算法、语言、规范三个科目，适合有一定基础的开发者
- **Pro级**：专家级，包含算法、语言、规范、设计四个科目，适合高级开发者

### 4.2 科目说明
- **科目1**：算法 - 数据结构、算法设计与分析等
- **科目2**：语言 - 编程语言语法、特性等
- **科目3**：规范 - 编码规范、最佳实践等
- **科目4**：设计 - 系统设计、架构设计等

## 5. 注意事项

1. **IP变动**：如果您的IP地址发生变动，系统会将您视为新用户，之前的学习数据将无法访问。请联系管理员进行数据迁移。

2. **考试时间**：系统会严格计时，请合理安排答题时间。

3. **答案保存**：系统会自动保存您的答案，但建议在完成题目后手动检查保存状态。

4. **错题消灭**：错题需要连续做对3次才能标记为已掌握，请认真复习。

5. **排行榜**：算法科目不参与排行榜排名，其他科目按得分排序。
